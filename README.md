# estatemanager
A WPF estate manager application coded in C# .NET

# SUJET
But : concevoir une application WPF de gestion des biens d’une agence immobilière.
## Fonctionnellement, au moins :
Permettre la consultation d’un portefeuille de biens immobiliers (liste, détails, tri, filtres)
Permettre la gestion d’un portefeuille de biens immobiliers (ajout, modification, suppression)
Chaque bien doit être associé à une photo et à une localisation
## Techniquement, au moins :
MVVM
Ressources
Styles
Template sur liste
Binding
Asynchronisme
